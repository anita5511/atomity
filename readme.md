# Atomity – Backend Engineering Intern Take-Home Assignment

Thank you for your interest in Atomity.

This repository contains the take-home assignment for candidates applying to the
**Backend Engineering Intern (Java / Spring Boot)** role.

This task is designed to evaluate how you think about backend systems, correctness,
and maintainable code. It is not about building a perfect solution.

## What this repository contains

- `TASK.md` – Full problem statement and technical requirements
- `SUBMISSION.md` – How to submit your solution

Please read both files carefully before starting.
